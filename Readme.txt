Como usar GIT

Primero siempre hacer un

git pull

para descargar actualizaciones que otros hayan hecho
segundo hacer un

git status

para ver el estado de tus archivos
poner

git add .

para agregar los archivos

git commit -m

para poner un comentario de lo que hiciste

git status para verificar

git push -u origin master

para subir tus cambios
